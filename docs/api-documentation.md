# API Documentation

Full interactive docs are auto-generated by FastAPI at `http://localhost:8000/docs` once the backend is running.

## POST /api/classify

**Request:** `multipart/form-data` with field `file` (JPEG/PNG/WebP image).

**Response:**
```json
{
  "waste_label": "Plastic",
  "confidence": 0.94,
  "all_scores": { "Plastic": 0.94, "Paper": 0.03, ... }
}
```

## POST /api/carbon-estimate

**Request body:**
```json
{ "waste_label": "Plastic", "weight_kg": 0.02, "location": "MY" }
```

**Response:**
```json
{
  "waste_label": "Plastic",
  "weight_kg": 0.02,
  "location": "MY",
  "estimates": [
    { "method": "recycling",     "co2e_kg": 0.0012, "unit": "kg CO2e" },
    { "method": "landfill",      "co2e_kg": 0.0054, "unit": "kg CO2e" },
    { "method": "incineration",  "co2e_kg": 0.0038, "unit": "kg CO2e" }
  ],
  "recommended_method": "recycling"
}
```

## GET /api/recommendation?waste_label=Plastic

**Response:**
```json
{
  "waste_label": "Plastic",
  "tips": ["Place in the recycling bin (blue bin).", "..."],
  "ranked_methods": ["recycling", "incineration", "landfill"]
}
```
